Cloudflare 주요 상품

성능·보안·개발을 하나의 플랫폼에서 — 필요한 만큼만 선택하세요

⚡

CDN & 성능 최적화

Cloudflare의 글로벌 CDN은 정적·동적 콘텐츠를 사용자와 가장 가까운 엣지 서버에서 제공해 로딩 속도를 획기적으로 줄여줍니다. Argo Smart Routing으로 최적 경로를 자동 선택합니다.

CDN Argo Routing 이미지 최적화 HTTP/3
🛡️

DDoS 방어

초대형 DDoS 공격도 자동으로 탐지·차단합니다. L3/L4/L7 계층 전반을 커버하며 공격량에 비례한 추가 비용 없이 무제한 방어를 제공합니다.

L3/L4/L7 자동 완화 무제한 방어
🔥

웹 방화벽 (WAF)

OWASP Top 10, 제로데이 취약점을 포함한 수천 개의 규칙으로 웹 애플리케이션을 보호합니다. 커스텀 규칙 생성과 ML 기반 이상 탐지로 정교한 공격도 막아냅니다.

OWASP 커스텀 규칙 ML 탐지 봇 관리
🌐

DNS & 도메인 관리

세계 최고 수준의 응답 속도를 자랑하는 권위 DNS 서버. DNSSEC, Anycast 라우팅, 분석 대시보드를 통해 도메인을 안정적으로 관리하세요.

권위 DNS DNSSEC Anycast
🔒

Zero Trust 보안

Cloudflare Access와 ZTNA로 VPN 없이도 내부 애플리케이션을 안전하게 보호합니다. SSO, MFA 연동과 세분화된 접근 정책으로 기업 보안을 강화하세요.

ZTNA Access SSO/MFA SASE
⚙️

Cloudflare Workers

서버리스 엣지 컴퓨팅 플랫폼. 전 세계 330+개 엣지에서 JavaScript/Rust/Python 코드를 실행하고 KV, R2, D1으로 데이터를 저장하세요. 콜드 스타트 없는 초고속 실행.

Serverless Edge Computing KV/R2/D1 AI Gateway
🌊

Cloudflare Spectrum

HTTP/HTTPS를 넘어 모든 TCP/UDP 기반 애플리케이션까지 DDoS 방어와 프록시 기능을 확장합니다. 게임 서버, 이메일, FTP, SSH 등 비웹 서비스도 Cloudflare 네트워크로 보호하세요.

TCP/UDP 보호 게임 서버 메일 서버 DDoS 방어 IP 은폐
⚖️

로드 밸런싱

Cloudflare Load Balancer로 글로벌 트래픽을 서버 상태에 따라 지능적으로 분산합니다. 실시간 헬스체크로 장애 서버를 자동 감지하고, 페일오버를 통해 서비스 중단 없이 트래픽을 우회합니다.

글로벌 LB 헬스체크 페일오버 지오 라우팅
📦

R2 오브젝트 스토리지

S3 호환 오브젝트 스토리지로 이그레스 비용이 없습니다. 대용량 파일 저장·배포를 AWS S3 대비 훨씬 저렴하게 운영하고 Workers와 완벽하게 연동됩니다.

S3 호환 무료 이그레스 Workers 연동
